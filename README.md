# Study Assistant

A full-stack Study Assistant web application built with Next.js, TypeScript, FastAPI, and SQLite.

This project helps users manage study tasks, generate AI-inspired study plans, and summarize notes through a modern dashboard interface.

---

## Features

### Task Management

- Create tasks
- Edit/update tasks
- Delete tasks
- Mark tasks as completed
- Search/filter tasks

### Study Plan

- Generate a simple AI-inspired study plan based on study days but only logic not real AI

### Notes Summarizer

- Summarize long notes into shorter text

### Modern UI

- Dashboard sidebar navigation
- Responsive design
- Confirmation dialogs
- Modern task cards
- Tailwind CSS styling

---
